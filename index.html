How to Generate Your Own
1

Install

Clone the repo and install the Python dependency.

2

Set API Key

Add your Anthropic API key as an environment variable.

3

Generate

Run generate.py with your topic. Claude researches and builds the full interactive mindmap.

4

Open & Share

Open the output HTML in any browser. No server needed — fully self-contained.

# Clone & install git clone https://github.com/parkpineventure/market-mindmap cd market-mindmap pip install -r requirements.txt # Set API key (Windows PowerShell) $env:ANTHROPIC_API_KEY = "sk-ant-..." # Generate a mindmap for any topic python generate.py "AI Infrastructure buildout" python generate.py "GLP-1 drug ecosystem" --output glp1.html python generate.py "Copper supply chain critical minerals" python generate.py "US defence tech after Ukraine"
Features
📡

Radial Supply Chain Map

D3.js interactive mindmap with upstream bottlenecks and downstream beneficiaries. Zoom, pan, expand/collapse sectors.

📋

Investment Brief Panel

Click any bubble to open a sidebar with public tickers, revenue, growth, market cap, investment signals, and a bull/bear thesis.

⤢

Fullscreen Mode

Expand any Investment Brief into a full-screen overlay with wider firm tables, private IPO watch, and detailed bull/bear cases.

🔍

Filter & Focus

Filter by upstream only, downstream only, critical bottlenecks, or highest-conviction plays with one click.

🏷️

Sector Role Labels

Every ticker includes a one-liner explaining exactly what that company does in the specific sector — not just who they are.

🤖

AI-Powered Research

Claude researches real revenue, growth rates, market caps, supply tensions, and writes investment theses from current data.
